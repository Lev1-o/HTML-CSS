Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>.

Nesta frase, tenho um termo em destaque usando a tag <strong>.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I>.

Nesta frase, temos um termo em ênfase usando a tag <em>.

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro, temos um texto que não herdou as características da outra mark.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando as tags <big> e <small>.

Texto deletado

Podemos marcar um texto como excluído para indicar que deve ser lido, mas não considerado. Apenas usando a tag <del>

Sublinhado / Texto inserido

Podemos marcar sublinhando um texto usando a tag <u>.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins>.

Texto Sobrescrito

Para inserir coisas do tipo: x20 + 3. Usa-se a tag <sup>

Texto Subscrito

Para inserir coisas do tipo: H2O. Usa-se a tag <sub>
